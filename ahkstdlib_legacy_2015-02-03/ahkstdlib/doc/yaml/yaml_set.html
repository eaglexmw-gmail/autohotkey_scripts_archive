Yaml_Set

Yaml_Set can be used to set a value for a key in a database.

Yaml_Set(pYaml, Key, Value)

Parameters

Parameter

Description
pYaml

This is the pointer previously returned from Yaml_Init().
Key

Key to set value for. You can use dot syntax as well.
Value

Value to store for above key.

Return Value

None.

Remarks

None.
