EmoPaC - Miranda IM Emoticon Pack Creator
------------------------------------------------------
by Drugwash, April 2008
--------------------------------

EmoPaC is a small program written in AutoHotkey language.
It is designed to create emoticon packs for use in Miranda IM
with Ricardo Pescuma Domenecci's Emoticons plug-in.

The code written by myself is in the public domain.
However, the program contains some code and a library
created by other people, so their work will fall under their own licensing terms.
