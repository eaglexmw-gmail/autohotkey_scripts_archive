Styles

QHTM supports all of the basic styles you could need:
  • Italic Italic
  • Bold Bold
  • Underline Underline
  • Strikethrough Underline
  • Superscript Superscript
  • Subcript Subcript

We can of course combine them:

  • Bold italic Bold italic
  • Bold Undeline Bold Undeline
  • Bold italic underline Bold italic underline
  • Italic underline Italic underline

Whilst not strictly a style we can use ACRONYM to define a URI (hover the mouse over it to see)
