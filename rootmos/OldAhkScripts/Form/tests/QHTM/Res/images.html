Image Support

QHTM Supports the following images formats:

  • BMP
  • MNG
  • PCX
  • JPEG
  • ICO
  • GIF You'll need a licence for GIF though
  • Anything else. QHTM has an image callback mechanism!

QHTM uses a custom library called ImgLib, which means you get the full source code to ImgLib too. ImgLib is a powerful and flexible tool for loading and displaying images and it can be used in your applications directly.

What else...
We could do the same with any image type. Not just BMP format images. This include JPG's and PNG. PNG supports native transparency so QHTM will also use that.

We can align images to the top, middle or bottom

We can also align images to the left, center or right. Left and right alignment means that really, really, really, really, really, really, really, really, really, really, really, really, really, really, really, long paragraphs will wrap around the image nicely.

PNG Images

PNG images natively support transparency and alpha channels. This gives us much more scope to be creative and to make our images blend with the background.

With an alpha channel:
- white background
- blue background
- green background
- red background
- a tiled image background
- red to white gradient background

And finally...
QHTM allows you to handle image loading and optionally handle the display. This is very powerful and incredibly flexible. It means you are free to decide how you want QHTM to acquire the images.
